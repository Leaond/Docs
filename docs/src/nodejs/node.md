## Buffer 模块
Buffer模块用于分配内存空间，Buffer身上的方法 from、alloc、allocunsafe

buffer类似于数组，可以用下标的方式来读取里面的数据。buffer每一位能存储的最大数字是255，当超过这个限制就会溢出，显示不全。同时中文存储的大小也不一样
